# FoxSay Backend

This directory is a FastAPI-oriented structural placeholder managed by `uv` and `pyproject.toml`.

No runtime dependencies are installed yet. The schema and API contract files exist to lock the MVP boundary before implementation.

## Required Runtime Configuration
- `DEEPSEEK_API_KEY`
- `DEEPSEEK_API_BASE`
- `DEEPSEEK_MODEL`
- `QDRANT_URL`
- `FOXSAY_ENV`

Do not hardcode credentials or real course materials.
